# WishFlix

## Contexte métier

WishFlix est une plateforme de catalogue de jeux vidéo permettant aux utilisateurs de découvrir, consulter et gérer leur liste de souhaits (wishlist) de jeux.

### Fonctionnalités principales

- **Catalogue de jeux** : Parcourir une collection de jeux vidéo avec filtres par catégorie (RPG, Action, Aventure)
- **Recherche** : Rechercher des jeux par titre ou mots-clés
- **Wishlist** : Ajouter/retirer des jeux à sa liste de souhaits personnelle
- **Fiches détaillées** : Consulter les informations complètes sur chaque jeu (synopsis, note, plateformes, année)
- **Authentification** : Système de connexion pour gérer son profil

### Pages disponibles

- `home.html` : Page d'accueil avec catalogue et hero section
- `game-detail.html` : Fiche détaillée d'un jeu
- `wishlist.html` : Liste personnelle des jeux souhaités
- `login.html` : Page de connexion
- `404.html` : Page d'erreur

### Technologies

- HTML5 / CSS3
- TailwindCSS + DaisyUI pour le design
- Architecture statique (prêt pour intégration)
